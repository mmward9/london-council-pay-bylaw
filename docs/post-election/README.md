# Post-election handoff materials

The PACT bylaw stays in this repo and on ward9.online pre-election. The materials in this directory are the working papers that would be used in a post-election Solicitor / Treasurer review cycle, if and when the bylaw advances to formal introduction. They are preserved here for transparency about what additional work the bylaw would require before a formal council vote.

This directory is not load-bearing for the current public-facing artifact. The bylaw text in `../../rendered/BYLAW.md` and the audit working papers in `../` stand on their own.

## Contents

| File | What it is |
|---|---|
| [solicitor-handoff.md](solicitor-handoff.md) | The original v12 pre-introduction handoff memo. Identifies items requiring City Solicitor and Treasurer confirmation. **Deferred to post-election per locked campaign decision.** Items requiring confirmation in a future cycle are also captured at the end of [`../v13-proposal.md`](../v13-proposal.md) under "Post-election handoff items," updated for the v13.2 mechanism. |
